## Run these commands in terminal:

```bash
# 1. Navigate to project folder
cd youtube-seo-analyzer

# 2. Initialize git (if not already)
git init

# 3. Add all files
git add .

# 4. Commit
git commit -m "ContentCraft AI - Complete project with credit system, admin panel, and golden theme"

# 5. Set main branch
git branch -M main

# 6. Add remote repository
git remote add origin https://github.com/algox27/ContentCraft-AI.git

# 7. Push to GitHub
git push -u origin main
```

## If you get "remote already exists" error:

```bash
git remote remove origin
git remote add origin https://github.com/algox27/ContentCraft-AI.git
git push -u origin main
```

## If you need to force push:

```bash
git push -u origin main --force
```
